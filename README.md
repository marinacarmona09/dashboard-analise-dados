# Dashboard de Análise de Dados

Este dashboard permite analisar salários na área de dados, considerando fatores como senioridade, tipo de contrato, localização e trabalho remoto.

## Tecnologias utilizadas
- Python
- Streamlit
- Pandas
- Plotly

## Preview do Dashboard
![Dashboard](dashboard.png)

## Estrutura do projeto
- `app.py` → aplicação principal do dashboard
- `requirements.txt` → bibliotecas necessárias
- `README.md` → documentação do projeto

## Funcionalidades
- Visualização de dados em gráficos interativos
- Análise de salários
- Filtros dinâmicos para exploração dos dados

## Dataset

Os dados utilizados neste projeto estão disponíveis no arquivo `dados-imersao.csv`. O dataset contém informações sobre profissionais da área de dados, incluindo cargo, salário, moeda, localização e tipo de contrato.

## Como executar o projeto

```bash
pip install -r requirements.txt
streamlit run app.py
```
